# Quiz Application

An interactive Quiz Web Application built using HTML, CSS, and JavaScript. The application allows users to answer multiple-choice questions and displays the final score at the end of the quiz.

## Features

* Multiple choice quiz questions
* Dynamic question display
* Instant answer selection
* Automatic score calculation
* Interactive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

quiz-app
│
├── index.html
├── style.css
└── script.js

## How It Works

1. The quiz displays one question at a time.
2. The user selects an answer from the available options.
3. The application moves to the next question.
4. After all questions are answered, the final score is displayed.
